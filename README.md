This is a basic terminal based Tic Tac Toe game written in C.
This project demonstrates basic concepts of C programming including arrays, functions, loops, conditionals, and input validation.

It features:
- Two-player mode (Player X vs Player O)
- Interactive command-line interface
- Input validation
- Automatic win and draw detection
- Clear and readable board display
